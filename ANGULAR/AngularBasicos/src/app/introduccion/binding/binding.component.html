Técnicas binding

Interpolación

Si tenemos una propiedad en el archivo .ts del componente, le podemos dar salida con las dobles llaves. También permite operaciones

Me llamo: {{nombre}}

{{n1*n2}}

La interpolación también puede afectar a las propiedades html que conocemos. En este ejemplo, vamos a poner una imagen mediante una propiedad cuyo valor va a estar gestionado por el archivo TypeScript

Pero si queremos hacer esto, hay otro método mejor (sugerido)

Property binding

El property binding permite vincular valores de propiedades html con valores de propiedades del archivo TypeScript. Es muy parecida a la interpolación, pero la propiedad la envolveremos entre corchetes
